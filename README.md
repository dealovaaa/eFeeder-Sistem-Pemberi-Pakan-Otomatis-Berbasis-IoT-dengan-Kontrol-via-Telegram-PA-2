# eFeeder-Sistem-Pemberi-Pakan-Otomatis-Berbasis-IoT-dengan-Kontrol-via-Telegram-PA-2
Proyek eFeeder menggunakan ESP32, motor DC, servo, sensor ultrasonik, relay, panel surya, aki, SCC, serta rangkaian PCB. Sistem terhubung ke bot Telegram untuk kontrol jarak jauh dan notifikasi. Semua komponen bekerja bersama untuk otomatisasi pemberian pakan ternak.
